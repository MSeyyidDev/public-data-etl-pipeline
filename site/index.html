Public Data ETL Pipeline

A reproducible Python ETL demo that extracts synthetic public-data feeds, validates row-level quality rules, writes SQLite and Parquet outputs, and produces an HTML report for analyst review.

Open generated report
3 domains: weather, traffic, sales
105k+ generated fact rows
5 loaders and quality modules
offline deterministic demo data

What reviewers should inspect

  • etl/pipeline.py for orchestration and stage metrics.
  • etl/quality/checker.py for reusable validation rules.
  • etl/cli.py for the Typer command surface.

How to reproduce

Run etl run-all, etl quality-report, and etl report. The generated HTML report is copied into this static site for GitHub Pages.
